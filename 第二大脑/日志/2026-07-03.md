## 💻 终端活动
- `npm install -g pnpm` → 全局安装 pnpm 包管理器
- `cd agri_platform && pnpm install` → 安装项目依赖
- `npm install gsap react-icons` → 安装 GSAP 动画库和 React Icons
- `pip install blind-watermark` → 安装盲水印 Python 库
- `/Library/Developer/CommandLineTools/usr/bin/python3 -m pip install --upgrade pip` → 升级 pip

- `curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash` → 安装 Hermes Agent
- `hermes` 相关命令 + `source ~/.zshrc` → 初始化 Hermes 配置
