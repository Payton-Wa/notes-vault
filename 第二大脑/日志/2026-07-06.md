## 今日学到的关键点（睡前填）
- **iOS Obsidian Git 不支持 SSH**：Transport 列表里只有 HTTPS，必须用 `https://github.com/...` clone
- **launchd Full Disk Access**：在系统设置→隐私→完全磁盘访问权限给 `/bin/bash` 打勾才能访问 Documents
- **git merge conflict 在 iCloud vault**：用 `git checkout --theirs` 接受远程版本，`git rebase --abort` 丢弃变基
- **双 vault 同步策略**：Public Vault 用 git pull，iCloud vault 直接从 Public Vault 复制文件（避免冲突）
