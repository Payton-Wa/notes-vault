## 今日学到的关键点（睡前填）
- **ip-api.com** 免费 IP 地理定位 API（45 req/min），返回国家/省份/城市/经纬度/ISP
- **Nominatim (OpenStreetMap)** 反向地理编码 API，经纬度→道路名+门牌号，1 req/sec
- **ip-api 与 Nominatim 串联**：先用 ip-api 获取 IP 的经纬度，再用 Nominatim 反查得到精确到道路的地址
- **launchd** macOS 定时任务系统：plist 配置 + `launchctl bootstrap gui/$uid` 在用户域加载
- **Obsidian Git 插件** `autoPullInterval` 设为 5 可每 5 分钟自动拉取
- **iOS Obsidian 不支持 SSH**，必须用 HTTPS 地址克隆仓库
- **sql.js** 的 `INSERT … ON CONFLICT DO UPDATE` 语法可用于数据归并
- **launchd 沙箱限制**：后台进程无法访问 `~/Documents`（Full Disk Access 权限），用 `/tmp` 作为 WorkingDirectory 绕过
