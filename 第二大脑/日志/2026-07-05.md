## 今日学到的关键点（睡前填）
- **ip-api.com** 免费 API：`fields=status,country,regionName,city,district,lat,lon,isp,org`
- **Nominatim OSM**：经纬度→精确地址，1 req/sec，必须加 User-Agent header
- **launchd exit code 126**：Permission denied，沙箱不允许访问 `~/Documents`
- **解决**：脚本放 `~/Library/LaunchAgents/`，WorkingDirectory 设 `/tmp`，Full Disk Access 给 `/bin/bash`
- **ip-api + Nominatim 串联架构**：先获取 lat/lon → 缓存到 ip_geo 表 → 逆向地理编码 → 写入 Markdown
